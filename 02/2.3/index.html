URI (Uniform Resource Identifier, Уніфікований ідентифікатор ресурсів)

  • схема — схема звернення до ресурсу,
  • логін і пароль — відповідно ім’я користувача і пароль для доступу до ресурсу
  • хост — повне доменне ім’я або IP-адреса ресурсу
  1. порт — порт, по якому буде проведено звернення до хоста
  2. шлях — використовується для уточнюючої вказівки місця знаходження ресурсу
  3. параметри — рядок параметрів виду параметр=значення
